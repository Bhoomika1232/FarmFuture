Project Title: Farm-Future

The Problem: Mention the "Debt Trap" and "Middlemen" issues in agriculture.

The Solution: Explain the "Pre-harvest Crowdfunding" concept.

Tech Stack: Flask, SQLite, Tailwind CSS, Scikit-Learn.

Key Features: * Farmer/Investor Dashboards.

QR Code Digital Receipts.

Crop Price Prediction using Linear Regression.
How to Run:
1. python -m venv venv
2. source venv/bin/activate
3. pip install -r requirements.txt
4. python app.py
